## Alterar o servidor
Antes de compilar, trocar o IP do servidor no arquivo Jogador.java na seguinte linha:

```
  private static final String ENDERECO_SERVIDOR = "INSIRA O IP DO SERVIDOR AQUI";
```

## Compilação
Compilar ambos os arquivos antes de rodar.

```
javac Servidor.java
javac Jogador.java
```

## Execução
Compilar ambos os arquivos antes de rodar.

```
java Servidor
java Jogador
```
